
# api 文档

## 备注
接口可以共用，提交修改可以共用一个，
也可以每个大类做一个大接口，用字段区分用户做的是哪个进件

## 入口

> 用户商户认证信息接口、状态（包含小微商户，个体工商户，企业商户）

## 小微商户

> 法人身份证信息提交、修改、返显

> 商户经营信息提交、修改、返显

> 结算账户信息提交、修改、返显

## 个体工商户

> 营业执照信息提交、修改、返显

> 法人身份证信息提交、修改、返显

> 商户经营信息提交、修改、返显

> 结算账户信息提交、修改、返显


## 企业商户

> 营业执照信息提交、修改、返显

> 法人身份证信息提交、修改、返显

> 商户经营信息提交、修改、返显

> 结算账户信息提交、修改、返显

## 签约产品

> 产品列表信息


## 其他

> 开户支行
