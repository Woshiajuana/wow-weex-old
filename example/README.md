## 操作命令

> npm run serve => 启动一个本地服务

> npm run dev   => 本地打包JS

> npm run build => 生产打包JS

> node cmd/version [string]  =>  设置版本号

> node cmd/log  [boolean]   =>  是否启用日志输出

> node cmd/release [string] =>  设置发布的环境

> node cmd/tree => 生成tree.json

> node cmd/ip [string]  =>  设置本地环境IP
